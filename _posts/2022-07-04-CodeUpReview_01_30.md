### 1) 10진수 유니코드(Unicode) 값으로 출력하기
예시) 
<br/>
```python
n = ord(input()) #입력받은 문자를 10진수 유니코드 값으로 변환한 후, n에 저장 
print(n)
``` 
ord()는 어떤 문자의 순서 위치(ordinal position) 값을 의미한다. 실제로 각각의 문자들에는 연속된 정수 값이 순서에 따라 부여 되어 있다. A:65, B:66, C : 67...
<br/>

ord(c) : 문자 c를 10진수로 변환한 값 
<br/>

컴퓨터로 저장되고 처리되는 모든 데이터들은 2진수 형태로 정수화되어야 하는데, 컴퓨터에 문자를 저장하는 방법으로 아스키코드나 유니코드가 자주 사용된다. 

<br/>
예를들어, 영문 대문자 'A'는 10진수 값 65로 표현하고, 2진수(binary digit)값 1000001로 바꾸어 컴퓨터 내부에 저장한다. 

<br/>
유니코드는 세계 여러 나라의 문자를 공통된 코드 값으로 저장할 때 사용하는 표준 코드이다. 

### 2) 16진 정수 입력받아 8진수로 출력하기 
16진수를 입력받아 8진수(octal)로 출력하기 
```python
a = input()
n = int(a,16) #입력된 a를 16진수로 인식해 변수 n에 저장 
print('%o'%n) #n에 저장되어 있는 값을 8진수(octal) 형태 문자열로 출력 
```
참고
<br/>
8진법은 한 자리에 8개(0 1 2 3 4 5 6 7)의 문자를 사용한다.
<br/>
8진수 10은 10진수 8, 11은 9, 12는 10...와 같다.
